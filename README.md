kwtSMS for Freshdesk integrates the kwtSMS SMS gateway with Freshdesk, enabling automatic and manual SMS notifications for ticket events, agent communications, and admin alerts directly from your Freshdesk account.

## Features

- Automatic SMS notifications on ticket create, status change, and agent reply
- Admin SMS alerts on new tickets and priority escalations
- Ticket sidebar for agents to send manual SMS from any ticket
- Dashboard showing balance, message stats, and gateway status
- SMS templates with English and Arabic support and dynamic placeholders
- SMS logs with filtering and pagination
- Daily cron sync for balance, sender IDs, and coverage data

## Installation

1. Install the app from the Freshworks Marketplace.
2. Enter your kwtSMS API credentials (username and password) in the app settings.
3. Select your preferred sender ID.

## Configuration

- Enable or disable the gateway and test mode from the app settings.
- Configure SMS templates for each ticket event type.
- Add admin alert recipient phone numbers.
